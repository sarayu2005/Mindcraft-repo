Python Arrays: Scenario Learning

Scenario 1: The Game Inventory (Indexing)

Goal: You are building an RPG. The player has a backpack (list). You need to equip specific items based on their slot number.

inventory[0]
# Computers start counting at 0!
inventory[2]
# This grabs the 3rd item
inventory[-1]
# Negative numbers count from the END

Scenario 2: The Playlist (Slicing)

Goal: You have a list of songs. You want to play a specific "chunk" (slice) of the playlist, not just one song.

playlist[0:2]
# Start at 0, STOP before 2 (Indices 0, 1)
playlist[1:4]
# Start at 1, STOP before 4 (Indices 1, 2, 3)
