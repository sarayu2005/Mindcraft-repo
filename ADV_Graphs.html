🕸️ Introduction to Graphs

A graph is a non-linear data structure consisting of vertices (nodes) and edges connecting them.

TERMINOLOGY:
• Vertex: Node entity.
• Edge: Connection between nodes.
• Directed: Edges have direction (A→B).
• Weighted: Edges have cost/distance.
📐 DATA LOG: Undirected Graph
A ---- B | | D ---- C

🔍 Traversal: BFS & DFS

Explores as deep as possible using a Stack (or recursion).
