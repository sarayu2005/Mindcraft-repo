The Return Value

Functions often do work and then pass the result back to you. This is called a return value.

def double_number(x):
   return x * 2

result = double_number(5) # result becomes 10

Think of it like a vending machine: You put money in (Parameter), and it gives a snack back (Return).

The Vending Machine

Call the function with money. It returns an item based on the amount.
