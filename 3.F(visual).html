Visual Python Functions

1. The Basic Processing Core

A basic function is a reusable block of code. You "call" it by name, it runs its internal code, and then finishes. It doesn't need inputs and doesn't give back a value.

def activate_alarm(): # This is the core's job print("ALARM! ALARM!") # Call the function activate_alarm()

2. The Core with Input Slots (Parameters)

Functions become powerful when you can give them data to work with. These inputs are called Parameters (the slots) and Arguments (the actual data you drop in).

def greet_user(name): # Uses the input data print("Welcome, " + name) # Drop "Neo" into the 'name' slot greet_user("Neo")
PRINT "Welcome " + name
Console: Waiting for input...

3. The Core with Output Chute (Return)

The most useful functions process inputs and then give back a result. The `return` keyword is the output chute. The function *becomes* the value it returns.

def add_numbers(a, b): # Process the inputs result = a + b # Send it out the chute return result # The function call is replaced by the result (30) final_score = add_numbers(10, 20)
