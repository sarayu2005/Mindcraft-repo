🌳 Introduction to Trees

A tree is one of the most important non-linear data structures in computer science. Unlike linear data structures (arrays, linked lists), trees organize data hierarchically.

🎯 What is a Tree?

A tree is a collection of nodes connected by edges:

  • One node is the root (topmost).
  • Every node (except root) has exactly one parent.
  • Nodes can have zero or more children.
  • No cycles allowed.

🔍 Binary Search Trees

A Binary Search Tree (BST) is a specialized structure optimized for searching.

🎯 The BST Protocol

RULESET:

1. Left Subtree: All values < Node Value.
2. Right Subtree: All values > Node Value.
3. Recursive: Applies to every node.

class TreeNode:

🔄 Traversals: Part 1

Scanning the tree requires specific algorithms.

1️⃣ Inorder (Left → Root → Right)

Effect: Returns nodes in SORTED order for BSTs. 
def inorder(node):
    if node is None: return
    inorder(node.left)
    print(node.value)
    inorder(node.right)

2️⃣ Preorder (Root → Left → Right)

Useful for copying trees or prefix expressions.
