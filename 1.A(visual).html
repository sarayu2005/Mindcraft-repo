Python Arrays (Lists)

1. What is an Array?

💊

Imagine a pill organizer or a shelf. An array (called a List in Python) is a variable that can store multiple items at once. Instead of creating item1, item2, item3, you make one list called items. 

items = ["Sword", "Shield", "Potion"]

2. The "Zero-Index" Rule

This is the most important rule for visual learners: Computers start counting at 0.

Visual Representation:
"Sword" Index 0
"Shield" Index 1
"Potion" Index 2

To get the "Sword", you ask for items[0].
To get the "Potion", you ask for items[2].
