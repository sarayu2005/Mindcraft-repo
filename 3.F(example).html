🏷️ Example 1: The ID Card Maker (Parameters)

Functions act like templates. You define the layout once, and pass different data (`name`, `role`) to fill the blanks.

def create_badge(name, role):
    print("--- ID BADGE ---")
    print("Name: " + name)
    print("Role: " + role)

💵 Example 2: The Tax Calculator (Return)

Often, you don't want to just print input; you want to transform it (Math) and get the result back.

def add_tax(price):
    tax = price * 0.10
    total = price + tax
    return total  # Sends value back

# The function becomes the number
final = add_tax(100)

🔒 Example 3: The Security Gate (Boolean Return)

Functions are great for checking rules. They return True or False to tell the system what to do next.

def check_password(input_pass):
    if input_pass == "1234":
        return True
    else:
        return False
